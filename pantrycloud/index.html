Jump Right In

Quickly add Pantry to your project using one of our easy-to-use client libraries.

const pantry = require('pantry-node')

const pantryId = 'xxxx-xxxx-xxxx-xxxx-xxxx'
const pantryClient = new pantry(pantryId)

const payload = {
  pending: ['walk the dog'],
  complete: ['finish homework'],
}

await pantryClient.basket.update('TodoList', payload)
